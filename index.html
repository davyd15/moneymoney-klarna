🛍 Klarna

Klarna für MoneyMoney einrichten

4 Schritte · einmalig · ca. 1 Minute

1

Klarna öffnen

Öffne die Klarna-App im Browser. Du bist dort vermutlich schon eingeloggt.

🌐 Klarna öffnen
Nach dem Klicken erscheint Klarna in einem neuen Tab. Komm danach hier zurück – diese Seite bleibt offen.
2

Konsole öffnen

Wechsle zum Klarna-Tab und drücke diese Tastenkombination:

Chrome:  Cmd + Alt + J
Safari:   Cmd + Alt + C

Safari: Falls nichts passiert → Menü „Entwickler" → „JavaScript-Konsole".
Noch kein Entwickler-Menü? Safari → Einstellungen → Erweitert → „Entwickler-Menü einblenden".

3

Befehl ausführen

Wechsle zum Klarna-Tab. Klicke ins Konsoleneingabefeld und stelle sicher, dass es leer ist. Dann:

Chrome:

copy(localStorage['@KLAPP:signIn:refreshToken'])

Drücke Enter → undefined als Ausgabe ist normal → Token ist in der Zwischenablage.

Safari:

prompt('',localStorage['@KLAPP:signIn:refreshToken'])

Drücke Enter → Fenster öffnet sich → Cmd+A → Cmd+C → Fenster schließen.

4

Token in MoneyMoney einfügen

Der Token ist jetzt in deiner Zwischenablage.
Wechsle zu MoneyMoney und füge ihn ein: Cmd + V → OK.

🎉

Einrichtung abgeschlossen!

Der Token erneuert sich ab jetzt automatisch bei jedem Sync.
